# KUYBACA

Kuybaca adalah aplikasi untuk meningkatkan minat literasi secara daring dan gratis.

## Usage

Clone repository ini dan ikuti langkah berikut.

```python
npm install
npm run start
```

Gunakan postman untuk testing api.
